# Vivado Scaffold

Use this directory for the V1 block design:

- Zynq UltraScale+ MPSoC PS.
- AXI DMA or VDMA path from PS DDR to HLS AXI4-Stream inputs.
- Shared `stereo_pipeline` HLS IP from `../../../common/hls/ip_repo`.
- AXI DMA or stream sink for depth and point cloud output.
- AXI-Lite control mapped to PS.

The scripts are placeholders for a Vivado 2021.2 batch flow and should be expanded once the AXU5EVB-E board preset or exported PS configuration is available.
