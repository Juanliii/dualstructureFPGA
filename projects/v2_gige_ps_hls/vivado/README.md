# Vivado Scaffold

Use this directory for the V2 block design:

- Zynq UltraScale+ MPSoC PS with PS Ethernet capture path.
- AXI DMA/VDMA between PS DDR and the shared HLS stereo pipeline.
- Shared `stereo_pipeline` HLS IP from `../../../common/hls/ip_repo`.
- Optional GPIO/timer outputs for camera/projector trigger timing.
