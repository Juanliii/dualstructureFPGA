# Vitis Scaffold

The V2 app is a Linux user-space controller for:

- GigE Vision camera discovery/configuration;
- Mono8 frame acquisition from two MV-CS013-60GN cameras;
- trigger and projector timing control;
- DMA transfer to/from the HLS IP.

Scripts target XSCT/Vitis 2021.2 and expect `xsa/v2_gige_ps_hls.xsa`.
