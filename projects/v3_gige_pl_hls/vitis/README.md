# Vitis Scaffold

The V3 app configures and monitors PL:

- local/remote camera IP and UDP ports;
- GVSP packet/frame parameters;
- HLS AXI-Lite parameters;
- trigger and projector timing;
- packet loss, disorder, and frame counters;
- result stream status.

Scripts target XSCT/Vitis 2021.2 and expect `xsa/v3_gige_pl_hls.xsa`.
